WORD COMMANDS

a: Add a word to the current puzzle
s: Select a word to move
S: Select all words to move matching a regular expression
d: Delete a word from the current puzzle
D: Delete all words matching a regular expression

MOVEMENT COMMANDS

h | Left Arrow: Move selected word to the left
j | Down Arrow: Move selected word downwards
k | Up Arrow: Move selected word upwards
l | Right Arrow: Move selected word to the right
r: Rotate selected word

SERIES COMMANDS

w: Save the current puzzle in a new json file
W: Overwrite a puzzle in the series
x: Delete a puzzle in the series
c: Change series
o: Load data from a puzzle
m: Move a puzzle to a different location
v: View a series

MISCELLANEOUS

q: Quit program
?: Open this help
